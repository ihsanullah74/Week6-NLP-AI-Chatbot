# Week 6 – NLP, Transformer Models & AI Chatbot

This repository is part of my roadmap to becoming a Machine Learning Engineer.

## Roadmap Goal
Learning Data Science and Machine Learning step by step.
Week 6 focuses on Natural Language Processing, text classification, and AI chatbot development.

## Dataset
SMS Spam Collection Dataset – 5,572 messages labeled as spam or ham.

## Model Performance
- Naive Bayes: 97.13%
- Logistic Regression: 94.98%
- SVM: 97.76% (best model)

## Technologies Used
- Python 3
- NLTK
- Scikit-learn
- Hugging Face Transformers
- Streamlit
- Pandas & NumPy
- Git & GitHub

## How to Run
1. Clone this repository
2. Install requirements: pip install -r requirements.txt
3. Run the app: streamlit run app.py
